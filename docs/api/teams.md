# Teams API

Agent teams coordinate multiple peer agents that share a TODO list and exchange
messages. Enable them via `include_teams=True` on
[`create_deep_agent`][pydantic_deep.agent.create_deep_agent]. See
[Teams](../advanced/teams.md) for the conceptual overview.
